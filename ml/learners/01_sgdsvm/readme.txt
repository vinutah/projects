%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
                MAIN COMMANDS
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
To Evaluate the Distances reported please run:
    source run.sh distance

To Evaluate the Accuracy on the test set please run:
    source run.sh test



%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
                Other Commands:
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
I already used to generate the transfomed data:
    source run.sh transform

To perform cross validation:
    source run.sh train_cv
(the reports are available in ./data/cv_results/)
(the weights are available in ./data/weights/)

To train the classifier using best rho and c use:
    source run.sh train
